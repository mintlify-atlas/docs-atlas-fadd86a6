## Overview

DigiForum is designed with accessibility at its core, providing a comprehensive suite of accessibility features to ensure the platform is usable by people with various physical disabilities. All accessibility settings are preserved across sessions using localStorage and can be toggled on/off with immediate visual feedback.

### Implementation

```typescript AccessibilityMenu.tsx:72-82
const increaseFontSize = () => {
  const newSize = Math.min(settings.fontSize + 10, 150);
  updateSettings({ fontSize: newSize });
  toast.success(`Tamaño de texto: ${newSize}%`);
};

const decreaseFontSize = () => {
  const newSize = Math.max(settings.fontSize - 10, 80);
  updateSettings({ fontSize: newSize });
  toast.success(`Tamaño de texto: ${newSize}%`);
};
```

<Note>

</Note>

## Text-to-Speech (Read Aloud)

### Overview

The text-to-speech feature uses the Web Speech API to read selected text aloud:

```typescript AccessibilityMenu.tsx:89-100
const toggleReadAloud = () => {
  const newState = !settings.readAloud;
  updateSettings({ readAloud: newState });
  
  if (newState) {
    toast.success("Lectura en voz alta activada. Selecciona texto para escucharlo.");
  } else {
    toast.success("Lectura en voz alta desactivada");
    window.speechSynthesis.cancel();
    setIsReading(false);
  }
};
```

### Speech Synthesis

When text is selected, it's read aloud using Spanish voice:

```typescript AccessibilityMenu.tsx:102-110
const speakText = (text: string) => {
  window.speechSynthesis.cancel();
  const utterance = new SpeechSynthesisUtterance(text);
  utterance.lang = "es-ES";
  utterance.rate = 0.9;
  utterance.onstart = () => setIsReading(true);
  utterance.onend = () => setIsReading(false);
  window.speechSynthesis.speak(utterance);
};
```

### Text Selection Handler

An event listener captures text selection:

```typescript AccessibilityMenu.tsx:122-140
useEffect(() => {
  const handleTextSelection = () => {
    const selectedText = window.getSelection()?.toString().trim();
    if (selectedText && selectedText.length > 0) {
      speakText(selectedText);
    }
  };

  if (settings.readAloud) {
    document.addEventListener("mouseup", handleTextSelection);
  } else {
    document.removeEventListener("mouseup", handleTextSelection);
  }

  return () => {
    document.removeEventListener("mouseup", handleTextSelection);
    window.speechSynthesis.cancel();
  };
}, [settings.readAloud]);
```

<Warning>
  The Web Speech API requires browser support. It works in Chrome, Edge, Safari, and most modern browsers, but may have limited support in older browsers.
</Warning>

### Speech Parameters

<ParamField path="lang" type="string" default="es-ES">
  Language setting for Spanish speech synthesis
</ParamField>

<ParamField path="rate" type="number" default="0.9">
  Speech rate (0.9 = slightly slower than normal for clarity)
</ParamField>

## Settings Persistence

### Loading Settings

Settings are loaded from localStorage on mount:

```typescript AccessibilityMenu.tsx:34-41
useEffect(() => {
  const saved = localStorage.getItem("accessibility-settings");
  if (saved) {
    const parsed = JSON.parse(saved);
    setSettings(parsed);
    applySettings(parsed);
  }
}, []);
```

### Saving Settings

Settings are automatically saved when changed:

```typescript AccessibilityMenu.tsx:65-70
const updateSettings = (updates: Partial<AccessibilitySettings>) => {
  const newSettings = { ...settings, ...updates };
  setSettings(newSettings);
  localStorage.setItem("accessibility-settings", JSON.stringify(newSettings));
  applySettings(newSettings);
};
```

### Applying Settings

All settings are applied immediately:

```typescript AccessibilityMenu.tsx:43-63
const applySettings = (newSettings: AccessibilitySettings) => {
  document.documentElement.style.fontSize = `${newSettings.fontSize}%`;
  
  if (newSettings.highContrast) {
    document.body.classList.add("high-contrast");
  } else {
    document.body.classList.remove("high-contrast");
  }

  if (newSettings.accessibleFont) {
    document.body.classList.add("accessible-font");
  } else {
    document.body.classList.remove("accessible-font");
  }

  if (newSettings.largeCursor) {
    document.body.classList.add("large-cursor");
  } else {
    document.body.classList.remove("large-cursor");
  }
};
```

## Best Practices

<Warning>
  Always test accessibility features with actual users who have disabilities. Automated testing and developer assumptions cannot replace real user feedback.
</Warning>

<Tip>
  Consider implementing keyboard shortcuts for quick access to accessibility features:
  - `Ctrl + Plus`: Increase font size
  - `Ctrl + Minus`: Decrease font size
  - `Ctrl + Alt + C`: Toggle high contrast
  - `Ctrl + Alt + S`: Toggle text-to-speech
</Tip>
