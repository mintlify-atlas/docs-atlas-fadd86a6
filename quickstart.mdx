## Welcome to DigiForum

This guide will walk you through everything you need to start connecting with the community. In just a few minutes, you'll be ready to share your story and engage with others.

<Steps>
  <Step title="Create Your Account">
    
### Registration Process

Navigate to the authentication page and select the **"Crear Cuenta"** (Create Account) tab.

<Tip>
You can also click "Crear Cuenta" from the homepage hero section or navigation bar.
</Tip>

#### Required Information

Fill out the registration form with the following required fields:

```typescript
// Required fields for registration
interface RegistrationData {
  fullName: string;        // Your complete name
  email: string;           // Valid email address
  username: string;        // Unique username
  password: string;        // Minimum 8 characters
  confirmPassword: string; // Must match password
}
```

#### Validation Rules

The platform ensures data quality with these validations:

- **Email Format**: Must be valid format (e.g., user@example.com)
- **Password Length**: Minimum 8 characters
- **Password Match**: Confirmation must match original
- **Unique Email**: Email cannot already be registered
- **Unique Username**: Username must be available

<Warning>
If validation fails, you'll see a clear error message explaining what needs to be corrected.
</Warning>

#### Sample Registration Code

Here's how the registration validation works behind the scenes:

```typescript
// Email validation
if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email)) {
  setError("Por favor ingresa un correo electrónico válido");
  return;
}

// Password length validation
if (password.length < 8) {
  setError("La contraseña debe tener al menos 8 caracteres");
  return;
}

// Password match validation
if (password !== confirmPassword) {
  setError("Las contraseñas no coinciden");
  return;
}
```

<Info>
Once registered, you'll see a success message and be automatically redirected to the login tab after 2 seconds.
</Info>

### Creating a Post

Share your story, ask a question, or contribute resources to the community.

#### Step-by-Step

**1. Open the Creation Form**

Find the "Crear Nueva Publicación" (Create New Post) card and click **"Comenzar"** (Begin).

**2. Fill Out Post Details**

<CardGroup cols={3}>
  <Card title="Title" icon="heading">
    Write a descriptive title for your post
  </Card>
  <Card title="Category" icon="tags">
    Select the most appropriate category from the dropdown

  <Card title="Content" icon="file-text">

**3. Post Structure**

```typescript
interface Post {
  id: string;              // Auto-generated
  title: string;           // Your title (required)
  category: string;        // Selected category (required)
  content: string;         // Your message (required)
  authorId: string;        // Your user ID (auto)
  authorName: string;      // Your display name (auto)
  createdAt: string;       // Timestamp (auto)
}
```

**4. Submit Your Post**

Click **"Publicar"** (Publish) to share with the community. Your post will appear immediately at the top of the feed.

<Info>
All fields (title, category, content) are required. You'll see an error if any are missing.
</Info>

#### Example Post Creation


  <Step title="Use Accessibility Features">
    
### Accessibility Menu

DigiForum includes powerful accessibility features to ensure everyone can use the platform comfortably.

<Note>
Find the floating accessibility button (gear icon) in the bottom-right corner of every page.
</Note>

#### Available Features

**1. Text Size Adjustment**

Increase or decrease font size in 10% increments:
- **Range**: 80% to 150%
- **Default**: 100%

```typescript
// Font size control
const increaseFontSize = () => {
  const newSize = Math.min(settings.fontSize + 10, 150);
  updateSettings({ fontSize: newSize });
};

const decreaseFontSize = () => {
  const newSize = Math.max(settings.fontSize - 10, 80);
  updateSettings({ fontSize: newSize });
};
```

**2. High Contrast Mode**

Toggle high contrast display for improved visual clarity:

```typescript
if (newSettings.highContrast) {
  document.body.classList.add("high-contrast");
} else {
  document.body.classList.remove("high-contrast");
}
```

**3. Text-to-Speech**

Enable "Lectura en voz alta" (Read Aloud) to hear text spoken:
- Select any text on the page
- The platform will read it aloud in Spanish
- Adjustable speech rate (0.9x speed)

```typescript
const speakText = (text: string) => {
  window.speechSynthesis.cancel();
  const utterance = new SpeechSynthesisUtterance(text);
  utterance.lang = "es-ES";
  utterance.rate = 0.9;
  window.speechSynthesis.speak(utterance);
};
```

**4. Accessible Font**

Switch to a dyslexia-friendly, easier-to-read font:

```typescript
if (newSettings.accessibleFont) {
  document.body.classList.add("accessible-font");
}
```

**5. Large Cursor**

Enable an enlarged cursor for easier tracking:

```typescript
if (newSettings.largeCursor) {
  document.body.classList.add("large-cursor");
}
```
